# Spooky Candy Quest

A Halloween-themed 2D platformer built with p5.js, featuring candy collectibles, enemies, obstacles, sound effects, and a victory flag.

## Features
- Halloween-themed platformer gameplay
- Candy collectibles and score tracking
- Enemies and canyon obstacles
- Platforms, particle effects, and sound effects
- Win and game-over states

## Controls
- Left Arrow: Move left
- Right Arrow: Move right
- Up Arrow: Jump

## Built With
- HTML
- JavaScript
- p5.js
- p5.sound

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Play the game.
